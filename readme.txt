Ein Elementor Widget für Shisha-Bar Menükarten mit Tabak-Sorten, Getränken und Snacks.

== Description ==

Shisha Menu ist ein benutzerfreundliches Elementor Widget, das speziell für Shisha-Bars und Lounges entwickelt wurde. 

= Features =

* Vollständig kompatibel mit Elementor Pro 3.30+
* Kategorien und Unterkategorien für perfekte Organisation
* Responsive Design für alle Geräte
* Vollständige Design-Kontrolle über Elementor
* Mehrsprachigkeit (i18n) Support
* Barrierefreiheit nach WCAG 2.1 Standards

= Anforderungen =

* WordPress 6.0 oder höher
* PHP 7.4 oder höher
* Elementor 3.30 oder höher

== Installation ==

1. Laden Sie das Plugin hoch oder installieren Sie es über GitHub
2. Aktivieren Sie das Plugin im WordPress Admin
3. Stellen Sie sicher, dass Elementor installiert ist
4. Finden Sie das "Shisha Menu" Widget in Elementor

== Frequently Asked Questions ==

= Funktioniert das Plugin mit Elementor Free? =

Ja, das Plugin funktioniert mit Elementor Free und Pro.

= Kann ich mehrere Menüs auf einer Seite haben? =

Ja, Sie können beliebig viele Shisha Menu Widgets auf einer Seite platzieren.
